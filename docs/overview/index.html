Overview

Discover what is API2HTML and what are the capabilities that will make you launch your next website in no-time.

API2HTML is a dynamic website builder that only requires you to code HTML templates focusing on the view. At the same time API2HTML will inject data from external API resources into these views. The template engine used to render the templates is the logic-less template engine mustache.

API2HTML requires no controllers or models as all the data is provided by external resources. To do so, the application uses a configuration.json file that simply defines the mapping between URLs, templates and API calls.

The workflow to create pages in your new site is very simple:

  • Pick the URL pattern, including any :variables you might need e.g: /special-deals/:category
  • Define the external API call that will feed this page. Reuse the variables.
  • Write the template and use {{data}} to use the API response. Access subfields using the dot . separator. E.g: {{ data.status }}
  • Start the server!
