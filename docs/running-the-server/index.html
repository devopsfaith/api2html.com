Running the API2HTML server

Instructions on how to start the API2HTML server to serve an example blog site.

In order to start the API2HTML server you will need to provide the configuration file that defines the website you want to run. As the configuration file references the location of the mustache templates you will need to have them in the disc too. We will use the blog example provided in the source code.

Pick a project

As we said, we will use the blog example provided in the source code. Just do:

cd examples/blog

Or choose another example.

Start the server

Being in the example folder or the project root you want to start run:

api2html serve -d -c config.json -p 8080

This will start the server in http://localhost:8080
