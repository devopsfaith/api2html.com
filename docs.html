API to HTML Documentation

API2HTML takes any data from your API and injects it in mustache templates so you can build websites without coding controllers and models.

Getting Started

The first you need to do is to download or clone the source code of API2HTMLapi2html

Download or clone

Once you have your code, you need to compile it from the terminal. Just cd in the directory you have cloned or downloaded and execute: 

make prepare

This command will download all the necessary dependencies, when it finishes the following command will build the binary for your platform:

make prepare

Now you should have the binary api2html in the folder

Configure and start the server

The program only needs a configuration file to run. Create a file named config.json with all the configuration inside, for instance:

{
  "robots": true,
  "sitemap": true,
  "static_txt_content": [
    "1234567890.txt"
  ],
  "pages":[
    {
      "name": "products",
      "URLPattern": "/products/:tag",
      "BackendURLPattern": "http://myapi.company.com:8080/tag/:tag",
      "Template": "tag",
      "CacheTTL": "3600s",
      "extra": {"foo":"bar"}
    },
    {
      "name": "product",
      "URLPattern": "/product/:product",
      "BackendURLPattern": "http://myapi.company.com:8080/product/:product",
      "Template": "product",
      "CacheTTL": "3600s"
    },
    {
      "name": "home",
      "URLPattern": "/",
      "BackendURLPattern": "http://myapi.company.com:8080/home",
      "Template": "home",
      "CacheTTL": "3600s",
      "extra": {"is_home":true}
    }
  ],
  "templates": {"home":"home.mustache","product":"product.mustache","tag":"tag.mustache"},
  "layouts": {"main":"main_layout.mustache"},
  "extra":{
    "lang": "en-US",
    "url_static": "https://cdn.mycompany.com"
  }
}

Now with the file created start the server

api2html serve -d -c config.json -p 8080

This will start the server in http://localhost:8080
